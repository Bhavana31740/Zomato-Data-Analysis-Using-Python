# Zomato Data Analysis Using Python

A beginner-friendly exploratory data analysis of restaurant data from Zomato, using pandas, matplotlib, and seaborn.

## Contents
- `Zomato_Data_Analysis_Annotated.ipynb` — the analysis notebook, with a markdown explanation before every step
- `Zomato-data-.csv` — the dataset (must stay in the same folder as the notebook)

## What's inside
- Cleaning the `rate` column (e.g. `4.1/5` → `4.1`)
- Checking data types and missing values
- Visualizing restaurant types, votes, ratings, and cost distribution
- Comparing ratings and restaurant types against online order availability

## How to run
1. Clone this repo
2. Make sure `Zomato-data-.csv` is in the same folder as the notebook
3. Open `Zomato_Data_Analysis_Annotated.ipynb` in Jupyter Notebook / JupyterLab / VS Code
4. Run all cells top to bottom

## Requirements
```
pandas
numpy
matplotlib
seaborn
```
Install with:
```
pip install pandas numpy matplotlib seaborn
```
